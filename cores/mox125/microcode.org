Form 1 Instruction Microcode
----------------------------

name   - instruction name
code   - form 1 opcode value
wA?    - writes to register A?
wB?    - writes to register B?
rA?    - reads register A?
rB?    - reads register B?
wm?    - writes to memory?
rm?    - reads from memory?
notes  - misc. comments
